# Sandbox and Viewer Conformance Tests

Test cases for **Goal 3: Viewer conformance requirements and security sandboxing model**.

This directory will contain tests covering:

- Origin isolation between bundles and the viewer host
- Enforcement of default-deny network access
- Blocking of restricted APIs (filesystem, surveillance APIs)
- CSP injection and enforcement
- Prevention of navigation/popup escapes from the sandboxed context

*No test cases yet — this goal is pending spec completion.*
